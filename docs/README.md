## What is Finance Tracker?

Finance Tracker is a comprehensive financial transaction management application designed to help individuals and small businesses:

- **Track** all financial transactions across multiple accounts
- **Categorize** transactions automatically using rules or AI
- **Visualize** spending patterns with charts and maps
- **Automate** categorization workflows with a visual builder
- **Analyze** financial data to make better decisions

## Key Capabilities

### 1. Multi-Account Management
Manage multiple bank accounts (checking, savings, credit cards) in one place with support for different currencies.

### 2. Intelligent Categorization
Three ways to categorize transactions:
- **Manual** - Assign categories directly
- **Rule-Based** - Create rules that automatically categorize
- **AI-Powered** - Use machine learning to intelligently categorize

### 3. Visual Workflow Builder
Design sophisticated categorization workflows with a drag-and-drop interface, combining AI and rule-based logic.

### 4. Comprehensive Analytics
View spending trends, category breakdowns, and geographic patterns through interactive visualizations.

### 5. CSV Import
Import transactions from your bank statements with intelligent column detection.

## Who Is This For?

### Personal Finance Users
Individuals who want to:
- Track spending across multiple accounts
- Understand where their money goes
- Create budgets and monitor expenses
- Identify unusual spending patterns

### Small Business Owners
Business owners who need to:
- Track business expenses by category
- Monitor cash flow across accounts
- Generate financial reports
- Automate transaction categorization

### Financial Analysts
Professionals who want to:
- Analyze spending patterns
- Create custom categorization rules
- Build automated workflows
- Export data for further analysis
